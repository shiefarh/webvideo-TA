Website ini merupakan website untuk tempat ujicoba tugas akhir Shiela Farah Diva. Dibangun sebuah rancangan website yang dihosting pada web server yang menyajikan konten video on demand. Pada sisi server akan dicoba pengaruh Amazon Cloudfront sebagai CDN terhadap QoS dari sisi klien.

Ceritanya ini website VoD


Terdapat beberapa skenario ujicoba.

  • Uji pengaruh CDN berdasarkan lokasi pengguna
  • Uji pengaruh CDN berdasarkan jenis konten
  • Uji pengaruh CDN berdasarkan koneksi/bandwidth internet

    • Detail setiap skenario dapat dilihat dengan klik halaman tiap skenario.
